# 🔍 User Behavior & Funnel Analysis
**internee.pk · Data Analytics Internship · Task 4**

---

## 📌 Objective
Analyze how applicants interact with internee.pk, identify where they drop off, and find the biggest bottlenecks in the application flow.

---

## 🗺️ Application Journey

```
1,000  →  Homepage
  720  →  Programs Page        (28.0% dropped)
  480  →  Internship Page      (33.3% dropped)
  310  →  Started Application  (35.4% dropped)
  180  →  Submitted            (41.9% dropped)
   95  →  Confirmed            (47.2% dropped)
```
**Overall Conversion Rate: 9.5%**

---

## 🔴 Biggest Bottleneck
**Submitted → Confirmed** with a 47.2% drop rate.
